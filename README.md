# uuid22
> uuid22, short, url friendly uuid module, contain: 0-9,a-z,A-Z,-,_


## Installation
Install via `npm`:

```
$ npm i uuid22
```


## Usage
Create a short uuid
``` js
var uuid22 = require('uuid22');
var id = uuid22();
// "id" length is 22
```
